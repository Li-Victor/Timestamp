Timestamp Microservice

  • Pass a unix timestamp or a natural language date as a parameter.
    This will return a JSON object of both the Unix timestamp and the natural language form of that date.

  • Pass now as a parameter.
    This will return a JSON boject of both the Unix timestamp and the natural language date of the current time.

  • Any other parameter will return a JSON object with value null for the unix and natural date properties

Example usage

http://localhost:3000/August%2007,%202017

http://localhost:3000/August 07, 2017

http://localhost:3000/1502064000

http://localhost:3000/now

Example output

{"unix":1502064000, "natural":"August 7, 2017"}
